# Меню "Наши достижения"

Веб-приложение для удобного хранения и быстрого доступа к полезным сайтам.  
Выполнено в виде красивой визуальной ширмы с тремя разделами.

## Основные возможности

- Три раздела: **Услуги**, **Маркетплейсы**, **Интернет-магазины**
- При наведении на иконку — появляется название сайта
- Клик по иконке — добавляет сайт в список «Выбранные»
- Клик по названию в выбранных — открывает сайт в новой вкладке
- Полноценная **админ-панель** с возможностью:
  - Добавлять новые сайты
  - Загружать иконки с компьютера
  - Редактировать существующие сайты
  - Удалять сайты

## Технологии

- **Frontend**: HTML + CSS + JavaScript
- **Backend**: Node.js + Express
- **База данных**: MySQL

## Установка

### 1. База данных
1. Откройте **phpMyAdmin** или любой MySQL-клиент
2. Создайте базу данных и выполните файл **`db.sql`**

### 2. Сервер
```bash
# 1. Установите зависимости
npm install

# 2. Запустите сервер
node server.js
```
Сервер будет доступен по адресу: http://localhost:3000

### 3. Открытие сайта

- Главная страница: index.html
- Админ-панель: admin.html

Структура файлов

- index.html — главная страница с меню
- admin.html — панель управления
- style.css — дизайн
- script.js — логика фронтенда
- server.js — сервер и API
- db.sql — структура базы данных

Важные замечания

- В файле server.js укажите свой пароль от MySQL
- Иконки хранятся в базе данных в формате base64
- Рекомендуется использовать изображения размером до 200–300 КБ

Как пользоваться

- Откройте админ-панель и добавьте нужные сайты
- На главной странице кликайте по иконкам, чтобы собирать полезные ссылки
- Используйте собранный список для быстрого перехода
